工具简介 Overview

ocgt-monitor 是一个独立的 OpenCode Go 套餐额度监控工具。它直接调用 OpenCode Go 的 RPC 接口查询额度，大部分功能与 ocgt 无关。只有 history 命令需要读取 ocgt 产生的 JSONL 日志文件。Most features work independently of ocgt — only "history" reads ocgt log files.

快速开始 Quick Start

1

打开终端 Open a terminal

按 Win 键，输入 cmd 回车。然后进入项目目录：Navigate to the project folder.

C:\Users\...> cd C:\Users\14050\Desktop\opencodego_tokens\ocgt-monitor
2

获取凭证 Gather credentials

登录 opencode.ai，按 F12 → Application → Cookies → 复制完整 Cookie 字符串。Workspace ID 从 URL 中获取（wrk_xxx 格式）。Copy Cookie from browser DevTools, find Workspace ID in URL.

3

配置向导 Run setup wizard

C:\Users\...\ocgt-monitor> .\ocgt-monitor config init

按提示输入 Cookie、Workspace ID，DeepSeek Key 可跳过。Follow the prompts to enter your credentials.

4

查额度 Check your quota

C:\Users\...\ocgt-monitor> .\ocgt-monitor quota

显示 Rolling / Weekly / Monthly 三条进度条。Shows plan usage with progress bars and reset timers.

Cookie 过期？重新复制后运行 config init 更新。Cookie expired? Re-run config init.

多账户管理 Multiple Accounts

配置文件保存在 C:\Users\<用户名>\.ocgt-monitor\config.json，每个用户独立。Each Windows user gets their own isolated config.

.\ocgt-monitor config add work       # 添加新账户
.\ocgt-monitor config use work       # 切换到该账户
.\ocgt-monitor config list           # 列出所有账户

分享给朋友 Sharing the Binary

!
发送 exe 前清空你的配置：Clear your config before sharing. 
.\ocgt-monitor config delete default
